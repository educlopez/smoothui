# Components

SmoothUI provides a comprehensive collection of UI components that are both beautiful and functional. Each component is built with accessibility in mind and includes smooth animations powered by Framer Motion.

## Component Categories

### Basic UI Components

Essential building blocks for any interface:

- **Button** - Versatile button with multiple variants
- **Card** - Flexible container for content
- **Dialog** - Modal dialogs and overlays
- **Accordion** - Collapsible content sections
- **Avatar** - User profile images with fallbacks
- **Badge** - Status indicators and tags

### Animated Components

Components with smooth, delightful animations:

- **Wave Text** - Text with wave animation effects
- **Animated Input** - Input fields with smooth focus animations
- **Animated Progress Bar** - Progress indicators with animations
- **Animated Tags** - Interactive tag collections
- **AI Branch** - Animated AI-themed components

## Features

- **Accessible** - Built with accessibility best practices
- **Customizable** - Easy to theme and modify
- **Type-safe** - Full TypeScript support
- **Performant** - Optimized animations and rendering
- **Responsive** - Works on all screen sizes

## Getting Started

To use any component, simply install it using our CLI:

```bash
npx smoothui@latest add button
```

Then import and use it in your code:

```tsx
import { Button } from "@/components/smoothui/button";

export function MyComponent() {
  return <Button>Click me</Button>;
}
```
