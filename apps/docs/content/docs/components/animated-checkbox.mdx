## Features

- Three animation variants: draw, pop, morph
- Draw variant strokes in the checkmark with pathLength animation
- Pop variant bounces the checkmark with spring physics
- Morph variant fills background first, then reveals the check
- Three sizes: sm (16px), md (20px), lg (24px)
- Indeterminate state support with horizontal line
- Controlled and uncontrolled modes
- Accessible with hidden native input and proper ARIA attributes
- Full keyboard accessibility
- Reduced motion support
