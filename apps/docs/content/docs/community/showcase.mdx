### What to Include

<BodyText>
When submitting your project, please provide:
</BodyText>

- **Project name** — The name of your project or website
- **URL** — A live link where others can see it in action
- **Screenshot** — A screenshot or preview image of your project
- **Description** — A brief description of the project and which SmoothUI components you used
- **Tech stack** — The frameworks and tools you used alongside SmoothUI
