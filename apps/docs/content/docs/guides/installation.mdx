## Using SmoothUI CLI (Recommended)

<BodyText>
The SmoothUI CLI provides an interactive way to browse and install components with automatic dependency resolution.
</BodyText>

<FeatureCard title="Features" variant="info">
- Interactive component picker with search
- Auto-detects package manager (npm, pnpm, yarn, bun)
- Auto-detects component paths and tsconfig aliases
- Shows dependency tree before installation
- Auto-installs npm dependencies
</FeatureCard>

<Tabs items={['pnpm', 'npm', 'yarn', 'bun']} defaultIndex={0}>

<Tab value="pnpm">

```bash
# Add a single component
pnpm dlx smoothui-cli add siri-orb

# Add multiple components
pnpm dlx smoothui-cli add siri-orb grid-loader animated-tabs

# Interactive mode - browse and select
pnpm dlx smoothui-cli add

# List all available components
pnpm dlx smoothui-cli list
```

</Tab>

<Tab value="npm">

## Using shadcn CLI

<BodyText>
SmoothUI is also an official shadcn registry, so you can install components using the shadcn CLI with the `@smoothui{:js}` namespace.
</BodyText>

<Tabs items={['pnpm', 'npm', 'yarn', 'bun']} defaultIndex={0}>

<Tab value="pnpm">

```bash
# Install a single component
pnpm dlx shadcn@latest add @smoothui/siri-orb

# Install multiple components
pnpm dlx shadcn@latest add @smoothui/rich-popover @smoothui/animated-input

# Install components with dependencies
pnpm dlx shadcn@latest add @smoothui/scrollable-card-stack
```

</Tab>

### Use Components

<BodyText>
Import and use the installed components in your React application:
</BodyText>

```tsx
import { SiriOrb } from "@/components/smoothui/ui/SiriOrb"
import { RichPopover } from "@/components/smoothui/ui/RichPopover"

export default function App() {
  return (
    <div>
      <SiriOrb size="200px" />
      <RichPopover />
    </div>
  )
}
```
